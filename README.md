# tencent-call-web
### 项目运行

```bash
# 克隆项目
git clone git@10.200.0.3:rfkj/tencent-call/tencent-call-web.git

# 进入项目
cd tencent-call

# 安装依赖
npm install

# 启动服务
npm run serve
```

### 构建

```bash
npm run build
```

### 代码格式

```bash
# 检查
npm run lint

# 修复
npm run lint -- --fix
```
